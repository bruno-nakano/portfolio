# Bruno Nakano Portfolio - Design Ideas

## Design Brief
Recreating a creative director's portfolio with a desktop OS metaphor. The original uses:
- Green gradient background (white to mint green)
- Draggable folder icons scattered on a canvas
- Black window panels for content
- Monospace/uppercase typography
- Two navigation modes: folder desktop view and bubble/circle nav

## Idea 1: Retro-Futurist OS Terminal

**Design Movement**: Early 2000s web OS meets Y2K aesthetic

**Core Principles**:
- Faithful recreation of the original's desktop OS metaphor
- Dashed borders on folder icons (like selection rectangles)
- Black content windows with monospace text
- Minimal color palette: green gradient, black, white

**Color Philosophy**:
- Background: radial/linear gradient from white (#ffffff) to mint green (#90ee90)
- Folder windows: pure black (#000000) with white text
- Accent: yellow (#ffff00) for hover states
- Text: black on light background, white on dark

**Layout Paradigm**:
- Full-screen canvas with absolutely positioned folder icons
- Folders scattered organically across the canvas
- Fixed header bar (top) and footer bar (bottom)
- Content windows appear as floating panels

**Signature Elements**:
- Dashed red/yellow border on folder icons
- Folder icon SVG (classic Mac-style folder)
- Monospace uppercase text labels

**Interaction Philosophy**:
- Folders open on click, revealing black content panels
- Panels can be dragged around the canvas
- Maximize/close buttons on panels
- INDEX button opens a full grid view of all projects

**Animation**:
- Panels appear with subtle fade-in
- Folders have hover scale effect
- Smooth dragging with cursor feedback

**Typography System**:
- IBM Plex Mono for all text (monospace)
- Uppercase for all labels
- Small font sizes (10-12px) for that retro terminal feel

## Idea 2: Swiss Brutalism

**Design Movement**: Swiss International Style meets Digital Brutalism

**Core Principles**:
- Grid-breaking asymmetric layouts
- Raw, unpolished aesthetic with intentional roughness
- Typography as primary visual element

**Color Philosophy**:
- Stark black and white with single accent color
- Green used sparingly as highlight only

**Layout Paradigm**:
- Rigid grid that intentionally breaks
- Folders aligned to invisible grid but offset

**Signature Elements**:
- Thick borders instead of dashed
- Oversized typography

**Interaction Philosophy**:
- Minimal animations
- Direct, functional interactions

**Animation**:
- None - static transitions only

**Typography System**:
- Helvetica Neue Bold for headers
- Courier New for body

## Idea 3: Skeuomorphic Desktop Revival

**Design Movement**: Mac OS 9 / Windows 98 nostalgia

**Core Principles**:
- Pixel-perfect recreation of the original
- Desktop metaphor with draggable windows
- Retro color palette

**Color Philosophy**:
- Mint green gradient background
- Classic folder icons
- Black terminal windows

**Layout Paradigm**:
- Free-form canvas with draggable elements
- Fixed top and bottom bars

**Signature Elements**:
- Classic folder icons
- Window chrome with minimize/maximize/close

**Interaction Philosophy**:
- Drag-and-drop everything
- Double-click to open

**Animation**:
- Window open/close animations
- Smooth drag feedback

**Typography System**:
- IBM Plex Mono throughout
